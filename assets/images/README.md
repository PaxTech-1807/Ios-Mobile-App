# Directorio de Imágenes

Coloca aquí todas las imágenes necesarias para la aplicación.

## Estructura de Carpetas:

### `splash/`
- **splash_logo.png** - Logo principal para la pantalla de inicio (Splash Screen)
- Logo del barbero que aparece en el círculo blanco

### `onboarding/`
- **onboarding_1.png** - Ilustración para "Explore Barber Shop in Your Area"
  - Barbero con lupa examinando a un cliente
- **onboarding_2.png** - Ilustración para "Book Your Favorite Hair Stylist Shop"
  - Dos personas en una mesa (manicure/servicio)
- **onboarding_3.png** - Ilustración para "Let's Make Your More Beautiful"
  - Estilista secando el cabello de una cliente

### `icons/`
- Iconos adicionales que puedas necesitar
- Iconos de redes sociales (si prefieres usar imágenes en lugar de iconos de Material)

## Formato Recomendado:
- **Formato**: PNG con fondo transparente (preferible) o JPG
- **Resolución**: 
  - Para imágenes de onboarding: 800x600px o superior
  - Para iconos: 256x256px o superior
  - Para splash: 512x512px o superior

## Nota:
Después de agregar las imágenes, ejecuta:
```
flutter pub get
```
